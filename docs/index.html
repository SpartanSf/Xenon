createProjectileSystem (source, options)
Creates a projectile system for a sprite

Parameters:

  • source table The source sprite
  • options table Configuration options table with these possible fields:
    • spritePath string Path to projectile sprite
    • speed number Projectile speed (default: 8)
    • lifetime number Projectile lifetime in seconds (default: 3)
    • offset number Spawn offset from source (default: 0)
    • layer number Render layer (default: source.layer + 1)

Returns:

    table The projectile system with Fire and Update methods
